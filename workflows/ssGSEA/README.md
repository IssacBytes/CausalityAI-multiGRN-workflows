# ssGSEA workflows

This folder is reserved for ssGSEA learning and reproducibility workflows.

Planned contents:

- Official GenePattern module call examples.
- GCT/GMT input preparation.
- Parameter-focused workflow for normalization, weighting exponent, minimum overlap, and combine mode.

The official ssGSEA GenePattern source is tracked separately as the root-level `ssGSEA-gpmodule` submodule.
